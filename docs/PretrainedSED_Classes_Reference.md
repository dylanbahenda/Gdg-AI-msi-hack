### Classes NOT Used (389)

The remaining **389 classes** are available in PretrainedSED but intentionally not mapped by the project. These include:

- **Musical instruments & singing**: Guitar, piano, drums, singing, choir, etc.
- **Environmental & background**: Rain, wind, thunder, fire, ocean, etc.
- **Wildlife**: Cats, frogs, insects, wild animals, birds (beyond the 3 used), etc.
- **Speech-related**: All speech classes except those filtered by PANNs
- **Industrial/mechanical**: Jackhammer, chainsaw, power tools, machinery, etc.
- **Vehicle-related**: Train, airplane, helicopter, motorcycle (beyond engines), etc.
- **Electronic/appliance**: Microwave, vacuum, washing machine, refrigerator, etc.
- **Musical/tonal**: Whistles, bells, chimes, tones, synthesizers, etc.
- **Liquid/water**: Splashing, pouring, boiling, water streams, etc.
- **And many others...**

The project was deliberately designed to focus on **discrete, impulsive, safety-critical events** (gunshots, screams, doors, glass, sirens) rather than continuous sounds or environmental ambience.

## How to Extend

To add more classes from the full 447 to the project:

1. **Add an entry to `SEDOntology.label_to_class_map`** in `SED_model.ipynb` cell 16:
   ```python
   "Desired AudioSet class": {"it": "Italian name", "en": "English name"},
   ```

2. **Add a threshold to `SEDOntology.label_to_threshold_map`**:
   ```python
   "Desired AudioSet class": 0.10,  # adjust per sensitivity needs
   ```

3. The class will automatically be detected and scored during inference via the `classes_to_use` filter in `PretrainedSEDAnalyzer.analyze_segments`.

To see what classes the framework is capable of detecting, cross-reference this full list against `https://research.google.com/audioset/explore.html` (AudioSet Explorer) for descriptions and example sounds.
